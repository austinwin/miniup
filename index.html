Table to API

Turn an idea or spreadsheet into a backend data store.

MiniUp gives static apps the missing backend. Design a table from a plain-language idea, import CSV, Excel, or JSON, then use the generated API from your MiniUp page, a ChatGPT-built app, an internal tool, or an automation.

  • Editable dashboard tables with insert, edit, delete, search, paging, and CSV export.
  • REST endpoints for list, create, update, and delete records.
  • Write keys and per-table method scopes so public reads and protected writes stay separate.
  • Canonical API URLs, schema URLs, record templates, and fetch examples for generated apps.

GIS and data delivery

Upload a data file. Get hosted dataset and ArcGIS-ready links.

MiniUp handles the server-side import and returns exact URLs your app, map, GIS workflow, DuckDB reader, or ArcGIS Online map can use immediately.

GeoJSON, shapefile ZIP, Parquet, or GeoParquet in.

MiniUp can return map-ready GeoJSON, ArcGIS Online-friendly GeoJSON, FeatureServer compatibility URLs, and layer metadata.

  • Upload CSV, Excel, JSON, GeoJSON, ZIP shapefiles, Parquet, or GeoParquet.
  • Use hosted dataset links in browser apps instead of relying on local file pickers.
  • Use window.MINIUP_DATASET bindings in generated HTML apps.
  • Use map-ready GeoJSON URLs for Leaflet and browser GIS apps.
  • Use the returned GeoJSON URL in ArcGIS Online by choosing GeoJSON.
  • Keep FeatureServer compatibility URLs for Web AppBuilder or custom ArcGIS clients.
